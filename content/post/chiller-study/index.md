## 📊 핵심 성능 지표

### 1) COP (성능계수, Coefficient of Performance)

**= 냉동능력(kW) / 소비동력(kW)**

- 높을수록 좋음
- 일반 왕복동: 2~4
- 스크류: 3~5
- 터보(원심식): 5~7
- 흡수식: 0.7~1.4
- 관련 변수: 증발온도↑ → COP↑, 응축온도↓ → COP↑

### 2) IPLV (부분부하 성능계수, Integrated Part Load Value)

**= 가중평균 효율 (100%/75%/50%/25% 부하)**

- 실제 운전 조건 반영
- 가중치: 1% / 42% / 45% / 12%
- 빌딩 냉동기는 연간 대부분 부분부하 운전

### 3) 냉동톤 (RT, USRT)

**= 냉동능력 단위**

- 1RT = 3,024 kcal/h = 3.517 kW
- 관련 변수: 냉수 유량 × 온도차(입출구)

### 4) kW/RT (소비전력원단위)

**= 소비동력 / 냉동능력**

- 낮을수록 효율 좋음
- 터보냉동기: 0.5~0.7 kW/RT
- 스크류: 0.7~1.0 kW/RT
- = 3.517 / COP

## 🔧 성능에 영향주는 변수

### 냉매 사이클

| 변수 | 설명 | 영향 |
|------|------|------|
| 증발온도/압력 | 냉매가 증발하는 온도 | ↑ 높을수록 COP↑ (but 냉방능력↓) |
| 응축온도/압력 | 냉매가 응축하는 온도 | ↓ 낮을수록 COP↑ |
| 과냉도 | 응축 후 추가 냉각 정도 | ↑ 크면 냉동효과↑ |
| 과열도 | 증발 후 추가 가열 정도 | 적정 유지 (너무 크면 효율↓) |
| 압축비 | 응축압력 / 증발압력 | 작을수록 효율↑ |

### 냉수 측

| 변수 | 일반값 | 비고 |
|------|--------|------|
| 냉수 입구온도 | 12°C | |
| 냉수 출구온도 | 7°C | |
| 냉수 유량 | 능력에 비례 | |
| 냉수 온도차(ΔT) | 5°C | |

### 냉각수 측

| 변수 | 일반값 | 비고 |
|------|--------|------|
| 냉각수 입구온도 | 32°C | 냉각탑 성능 의존 |
| 냉각수 출구온도 | 37°C | |
| 냉각수 유량 | | 응축기 성능에 영향 |
| 외기 습구온도 | | 냉각탑 성능 → 냉각수 온도 결정 |

### 기계적 요소

| 변수 | 선택지/영향 |
|------|-------------|
| 압축기 종류 | 왕복동 / 스크류 / 스크롤 / 터보 |
| 용량제어 방식 | 언로더 / 인버터 / IGV |
| 냉매 종류 | R-134a, R-410A, R-32 등 |
| 열교환기 오염도(파울링) | ↑ 오염 → 효율↓ |

## 📐 변수 간 관계 정리

```
냉각수 온도↓ → 응축온도↓ → 압축비↓ → COP↑
냉수 온도↑   → 증발온도↑ → 압축비↓ → COP↑
부분부하     → 인버터 있으면 효율 유지, 없으면 효율↓
파울링↑      → 열교환 효율↓ → 응축온도↑ → COP↓
```

## 🛠️ 성능 개선 방법

### 🔽 응축온도 낮추기 → COP↑

- 냉각탑 용량 키우기 (팬 증속, 셀 추가)
- 냉각수 유량 늘리기 (펌프 병렬 운전, 인버터 제어)
- 냉각수 배관 길이 줄이기 (저항 감소 → 유량 확보)
- 응축기 전열관 스케일 세정 (파울링 제거)
- 응축기 전열면적 큰 장비 선정
- 냉각탑 충전재 교체/청소
- 외기 습구온도가 낮은 시간대에 축냉 운전

### 🔼 증발온도 높이기 → COP↑

- 냉수 설정온도 올리기 (7→8°C, 부하 허용 시)
- 증발기 전열관 청소 (파울링 제거)
- 증발기 전열면적 큰 장비 선정
- 냉수 유량 적정화 (너무 적으면 증발온도↓)
- 코일 면적 늘리기 (AHU/FCU) → 냉수 온도 높여도 냉방 가능

### 🔽 압축비 줄이기 → COP↑

- 위 두 가지의 조합 (응축온도↓ + 증발온도↑)
- 다단 압축 (2단 압축 + 중간냉각)
- 이코노마이저 적용

### 🔼 과냉도 확보 → 냉동효과↑

- 과냉각기(서브쿨러) 설치
- 응축기 하부 액냉매 체류 공간 확보
- 냉각수 온도를 더 낮추기

### 🔼 부분부하 효율 → IPLV↑

- 인버터 압축기 적용 (용량 가변)
- 터보냉동기 IGV(입구 가이드 베인) 제어
- 다수 냉동기 대수 제어 (순차 기동/정지)
- VWV(변유량) 시스템 적용 (냉수/냉각수)
- 냉각탑 팬 인버터 제어

### 🔽 배관/펌프 동력 줄이기 → 시스템 COP↑

- 배관 길이 최소화 (장비 배치 최적화)
- 배관 사이즈 적정 선정 (유속 2~3m/s)
- 펌프 병렬 운전 (부분부하 시 1대 정지)
- 펌프 인버터 적용 (차압 제어)
- 2차 펌프 시스템 적용 (1차: 정유량, 2차: 변유량)

### 🔽 파울링 방지 → 효율 유지

- 수처리(약품 투입) — 스케일/부식 방지
- 자동 볼 세정 장치 (응축기 전열관)
- 정기 화학 세정
- 스트레이너 청소
- 냉각수 블로우다운 관리

### 🌡️ 공조기(AHU/FCU) 측

- 코일 열수 늘리기 → 전열면적↑ → 냉수 고온 가능 → 증발온도↑
- 코일 핀 피치 적정화 + 청소
- 풍량 적정화 (VAV)
- 외기 도입량 최적화 (전열교환기 적용)

## 💡 파울링(Fouling)이란?

열교환기 전열관 표면에 이물질이 쌓이는 현상. 쉽게 말하면 **"배관 안쪽에 때가 끼는 것"**

### 종류

| 종류 | 설명 |
|------|------|
| 스케일(Scale) | 칼슘, 마그네슘 등 경수 성분이 굳어서 달라붙음 (주전자 안 하얀 때랑 같음) |
| 슬라임(Slime) | 미생물/조류가 번식해서 끈적한 막 형성 |
| 부식(Corrosion) | 금속 표면 산화로 녹 발생 |
| 이물질 | 모래, 먼지 등 외부 유입물 |

### 왜 문제인가

- 전열관에 때가 끼면 **열전달 저항**이 커짐
- 같은 냉각 효과를 내려면 더 많은 에너지 필요
- **응축온도↑ → COP↓**
- 심하면 배관 막힘, 장비 손상

### 특히 냉각수 측이 심한 이유

- 냉각탑이 개방형이라 외기 먼지/미생물 유입
- 물이 증발하면서 미네랄 농축 → 스케일

### 대응

- 수처리(약품) — 스케일/부식 방지
- 자동 볼 세정 장치
- 정기 화학 세정
- 냉각수 블로우다운 관리
