## 📊 핵심 성능 지표

### 1) COP (성능계수, Coefficient of Performance)

**= 냉동능력(kW) / 소비동력(kW)**

- 높을수록 좋음
- 일반 왕복동: 2~4
- 스크류: 3~5
- 터보(원심식): 5~7
- 흡수식: 0.7~1.4
- 관련 변수: 증발온도↑ → COP↑, 응축온도↓ → COP↑

### 2) IPLV (부분부하 성능계수, Integrated Part Load Value)

**= 가중평균 효율 (100%/75%/50%/25% 부하)**

- 실제 운전 조건 반영
- 가중치: 1% / 42% / 45% / 12%
- 빌딩 냉동기는 연간 대부분 부분부하 운전

### 3) 냉동톤 (RT, USRT)

**= 냉동능력 단위**

- 1RT = 3,024 kcal/h = 3.517 kW
- 관련 변수: 냉수 유량 × 온도차(입출구)

### 4) kW/RT (소비전력원단위)

**= 소비동력 / 냉동능력**

- 낮을수록 효율 좋음
- 터보냉동기: 0.5~0.7 kW/RT
- 스크류: 0.7~1.0 kW/RT
- = 3.517 / COP

## 🔧 성능에 영향주는 변수

### 냉매 사이클

| 변수 | 설명 | 영향 |
|------|------|------|
| 증발온도/압력 | 냉매가 증발하는 온도 | ↑ 높을수록 COP↑ (but 냉방능력↓) |
| 응축온도/압력 | 냉매가 응축하는 온도 | ↓ 낮을수록 COP↑ |
| 과냉도 | 응축 후 추가 냉각 정도 | ↑ 크면 냉동효과↑ |
| 과열도 | 증발 후 추가 가열 정도 | 적정 유지 (너무 크면 효율↓) |
| 압축비 | 응축압력 / 증발압력 | 작을수록 효율↑ |

### 냉수 측

| 변수 | 일반값 | 비고 |
|------|--------|------|
| 냉수 입구온도 | 12°C | |
| 냉수 출구온도 | 7°C | |
| 냉수 유량 | 능력에 비례 | |
| 냉수 온도차(ΔT) | 5°C | |

### 냉각수 측

| 변수 | 일반값 | 비고 |
|------|--------|------|
| 냉각수 입구온도 | 32°C | 냉각탑 성능 의존 |
| 냉각수 출구온도 | 37°C | |
| 냉각수 유량 | | 응축기 성능에 영향 |
| 외기 습구온도 | | 냉각탑 성능 → 냉각수 온도 결정 |

### 기계적 요소

| 변수 | 선택지/영향 |
|------|-------------|
| 압축기 종류 | 왕복동 / 스크류 / 스크롤 / 터보 |
| 용량제어 방식 | 언로더 / 인버터 / IGV |
| 냉매 종류 | R-134a, R-410A, R-32 등 |
| 열교환기 오염도(파울링) | ↑ 오염 → 효율↓ |

---

## 📐 변수 간 관계 정리

```
냉각수 온도↓ → 응축온도↓ → 압축비↓ → COP↑
냉수 온도↑   → 증발온도↑ → 압축비↓ → COP↑
부분부하     → 인버터 있으면 효율 유지, 없으면 효율↓
파울링↑      → 열교환 효율↓ → 응축온도↑ → COP↓
```
