HVAC 열역학 스터디 — Day 10
엔탈피 (h = u + Pv)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

오늘은 냉동공조 분야에서 가장 많이 쓰이는 물리량,
엔탈피(enthalpy)를 집중적으로 다룹니다.
P-h 선도까지 맛보기로 살펴볼 거예요!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[1] 엔탈피란
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

엔탈피(H)의 정의:
H = U + PV

비엔탈피(단위 질량당):
h = u + Pv

여기서
- u: 비내부에너지 (kJ/kg)
- P: 압력 (kPa)
- v: 비체적 (m3/kg)
- Pv: 유동일 (flow work) — 유체가 흐르면서 하는 일

쉽게 말하면:
엔탈피 = 물질이 가진 내부에너지 + 그 물질이 자리를
차지하면서 주변을 밀어내는 데 쓰는 에너지

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[2] 왜 엔탈피를 쓰는가
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

개방계(냉매가 흐르는 시스템)에서는
유체가 들어오고 나가면서 유동일(Pv)을 동반합니다.

만약 내부에너지(u)만 쓰면
매번 유동일을 따로 계산해야 해서 복잡합니다.

그래서 u + Pv를 묶어서 h라고 정의하면:
Q = m x (h2 - h1)  ← 깔끔!

개방계 제1법칙이 엔탈피 하나로 정리됩니다.
이것이 냉동공조에서 엔탈피를 쓰는 핵심 이유입니다.

정압과정에서:
Q = DH (가한 열 = 엔탈피 변화)

어제 배운 내용과 연결되죠?
정압과정이 많은 공조 분야에서 엔탈피가 만능인 이유!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[3] 비엔탈피 (h, kJ/kg)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

실무에서는 항상 단위 질량당 엔탈피(h)를 씁니다.

냉매 물성표에서 바로 읽을 수 있어요:
(R-410A 예시)
- 증발온도 5도, 포화증기: h_g ≒ 423 kJ/kg
- 응축온도 40도, 포화액: h_f ≒ 260 kJ/kg
- 과열증기(압축 후): h ≒ 460 kJ/kg (압축비에 따라 다름)

공기의 비엔탈피:
습공기선도에서 읽는 그 "엔탈피"가 바로 이것!
- 건구온도 26도, 상대습도 50%: h ≒ 53 kJ/kg(건조공기)
- 외기 33도, 상대습도 60%: h ≒ 85 kJ/kg(건조공기)
- 차이가 곧 외기 부하 계산의 기초!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[4] 냉동 사이클에서 엔탈피의 의미
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

냉동 사이클의 4개 기기를 엔탈피로 분석하면:

(1) 압축기: h1 → h2 (엔탈피 증가)
    압축기 일 W = h2 - h1

(2) 응축기: h2 → h3 (엔탈피 감소)
    방열량 Qc = h2 - h3

(3) 팽창밸브: h3 → h4 (엔탈피 변화 없음!)
    등엔탈피 과정: h3 = h4

(4) 증발기: h4 → h1 (엔탈피 증가)
    냉동효과 Qe = h1 - h4

에너지 보존 확인:
Qe + W = Qc
(증발기 흡열 + 압축기 일 = 응축기 방열)

이것이 바로 제1법칙의 냉동 버전입니다!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[5] P-h 선도 맛보기
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

P-h 선도 = 압력(세로축) vs 엔탈피(가로축) 그래프

냉동 사이클을 P-h 선도에 그리면:

         고압 ----[응축기]----
              |              |
         [압축기]        [팽창밸브]
              |              |
         저압 ----[증발기]----

각 과정의 방향:
- 압축(1→2): 아래에서 위로, 오른쪽으로 (압력, 엔탈피 증가)
- 응축(2→3): 위쪽 수평선, 왼쪽으로 (압력 일정, 엔탈피 감소)
- 팽창(3→4): 위에서 아래로, 수직 (엔탈피 일정, 압력 감소)
- 증발(4→1): 아래쪽 수평선, 오른쪽으로 (압력 일정, 엔탈피 증가)

핵심 포인트:
가로축(엔탈피)에서 각 구간의 길이가 곧 에너지!
- 증발기 구간 길이 = 냉동효과
- 압축기 구간 길이 = 압축일
- 응축기 구간 길이 = 방열량

P-h 선도 하나로 냉동 사이클 전체를 파악할 수 있습니다.
(나중에 더 자세히 다룰 예정이에요)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[6] 냉동효과 = 증발기 엔탈피 차이
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

냉동효과(qe):
qe = h1 - h4 (kJ/kg)

냉동능력(Qe):
Qe = m x (h1 - h4) (kW)

실전 계산 예시:
R-410A 냉동기
- 증발기 출구(압축기 입구) h1 = 423 kJ/kg
- 팽창밸브 출구(증발기 입구) h4 = 260 kJ/kg
- 냉매 유량 m = 0.3 kg/s

냉동효과 qe = 423 - 260 = 163 kJ/kg
냉동능력 Qe = 0.3 x 163 = 48.9 kW = 13.9 USRT

COP = Qe / W = (h1 - h4) / (h2 - h1)
h2 = 460이면
COP = 163 / (460 - 423) = 163 / 37 = 4.4

COP 4.4면 실무에서 괜찮은 수준이죠!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Quiz (3문제)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q1. 엔탈피 h = u + Pv에서 Pv를 무엇이라 하며,
    왜 개방계에서 이것을 포함해야 하는가?

Q2. 냉동 사이클에서 팽창밸브를 통과할 때
    엔탈피는 어떻게 변하는가? (증가/감소/변화없음)
    그 이유는?

Q3. 증발기 입구 엔탈피 250 kJ/kg, 출구 420 kJ/kg,
    냉매 유량 0.2 kg/s일 때 냉동능력(kW)은?

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Quiz 정답
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

A1. Pv를 유동일(flow work)이라 한다.
    개방계에서는 유체가 흐르면서 앞의 유체를
    밀어내는 일이 발생하므로, 이 유동일까지
    포함해야 에너지 수지가 맞는다.
    그래서 u만이 아닌 h = u + Pv를 사용한다.

A2. 변화없음 (등엔탈피 과정, h3 = h4)
    팽창밸브에서는 열 출입(Q)도 없고 외부 일(W)도 없다.
    단지 압력만 떨어지면서 일부가 기화(플래시 가스)되고
    온도가 내려간다. 엔탈피 자체는 보존된다.

A3. Qe = m x (h_out - h_in)
    = 0.2 x (420 - 250)
    = 0.2 x 170
    = 34 kW (약 9.7 USRT)
