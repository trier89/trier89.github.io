━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Day 13 - Week 2 복습: 핵심 정리 + 종합 퀴즈
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

안녕하세요! 오늘은 Week 2(Day 8~12)에서 배운 내용을 빠르게 정리하는 시간입니다. 시험장에 들어가기 전 마지막으로 훑어보는 느낌으로 읽어주세요.

━━━ 1. 열역학 제1법칙 ━━━

핵심 한 줄: 에너지는 생성되거나 소멸되지 않고, 형태만 바뀐다.

Q = ΔU + W

- Q: 시스템에 가해진 열량
- ΔU: 내부에너지 변화
- W: 시스템이 한 일

공조냉동 실무에서는 "열은 어디서 왔고 어디로 갔는가?"를 추적하는 기본 도구입니다. 냉동기에서 증발기가 흡수한 열 + 압축기가 한 일 = 응축기에서 방출한 열, 이게 바로 제1법칙의 적용이에요.

━━━ 2. 엔탈피 (h = u + Pv) ━━━

핵심 한 줄: 유동하는 유체의 에너지를 하나로 묶은 것이 엔탈피다.

- u: 내부에너지, Pv: 유동일(flow work)
- 개방계(냉동 시스템 대부분)에서는 엔탈피가 핵심 변수
- P-h 선도의 가로축이 바로 이 엔탈피!
- 단위: kJ/kg

━━━ 3. 엔트로피 ━━━

핵심 한 줄: 에너지의 "질"을 나타내는 척도, 비가역성의 지표.

- 가역과정: 엔트로피 변화 = 0
- 비가역과정: 엔트로피는 항상 증가
- 실제 압축기는 비가역 → 이상적 압축보다 엔트로피 증가
- 기술사 시험에서 "엔트로피 증가 = 손실 증가"로 연결하면 됩니다

━━━ 4. 카르노 사이클 ━━━

핵심 한 줄: 이론적으로 가능한 최고 효율의 사이클 (현실엔 없음).

냉동 COP_카르노 = T_L / (T_H - T_L) [절대온도 K]

- 두 열원의 온도차가 작을수록 COP가 높다
- 실제 냉동기 COP는 항상 카르노보다 낮다
- "왜 카르노에 도달 못하나?" → 마찰, 압력손실, 열전달 온도차 등 비가역 요소 때문

━━━ 전체 연결 포인트 ━━━

제1법칙(에너지 보존) → 엔탈피(에너지 계산 도구) → 엔트로피(효율 판단 도구) → 카르노(이상적 한계)

이 흐름을 머릿속에 넣어두면 냉동 사이클 분석의 뼈대가 완성됩니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
종합 퀴즈 (5문제)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q1. 냉동기에서 증발기 흡열량 50kW, 압축기 입력 15kW일 때, 응축기 방열량은?

Q2. 엔탈피 h = u + Pv에서 Pv가 의미하는 것은 무엇이며, 왜 냉동 시스템 분석에서 내부에너지(u) 대신 엔탈피(h)를 쓰는가?

Q3. 실제 압축기에서 엔트로피가 증가하는 이유를 2가지 이상 서술하시오.

Q4. 증발온도 -10도C, 응축온도 40도C일 때 카르노 냉동 COP는?

Q5. 실제 냉동기의 COP가 카르노 COP보다 항상 낮은 이유를 제1법칙과 제2법칙 관점에서 각각 설명하시오.

━━━ 정답 ━━━

A1. 제1법칙 적용: Q_응축 = Q_증발 + W = 50 + 15 = 65kW
에너지는 사라지지 않습니다. 증발기에서 흡수한 열 + 압축기가 투입한 에너지 = 응축기에서 버려야 할 열.

A2. Pv는 유동일(flow work)으로, 유체가 흐르면서 밀어내는 데 필요한 에너지입니다. 냉동 시스템은 냉매가 연속적으로 흐르는 개방계이므로, 유동일을 포함한 엔탈피가 에너지 수지 계산에 더 적합합니다.

A3. (1) 냉매와 실린더 벽 사이의 마찰 (2) 압축 과정 중 냉매와 주변 간 열교환 (3) 밸브 및 배관에서의 압력손실. 이런 비가역 요소들이 엔트로피를 증가시킵니다.

A4. T_L = -10 + 273 = 263K, T_H = 40 + 273 = 313K
COP = 263 / (313 - 263) = 263 / 50 = 5.26

A5. 제1법칙 관점: 마찰열, 압력손실 등으로 압축기에 더 많은 일을 투입해야 하므로 COP 감소.
제2법칙 관점: 비가역 과정(유한 온도차 열전달, 마찰 등)이 엔트로피를 생성하여 사이클 효율이 카르노 한계 이하로 떨어짐.
