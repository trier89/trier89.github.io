HVAC 열역학 스터디 — Day 12
카르노 사이클

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

오늘은 열역학의 꽃, 카르노 사이클입니다!
이론적 최고 효율의 기준이 되는 사이클이라
실무에서도 시험에서도 빠지지 않는 핵심 주제예요.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[1] 카르노 사이클이란
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

카르노 사이클 = 두 열원 사이에서 작동하는
이론적으로 가장 효율이 높은 가역 사이클

특징:
- 모든 과정이 가역 (마찰 없음, 무한소 온도차 열전달)
- 현실에 존재하지 않지만, 효율의 "상한선" 역할
- 어떤 실제 냉동기도 카르노 COP를 넘을 수 없음

프랑스 과학자 사디 카르노(1824)가 제안했고,
이것이 열역학 제2법칙의 발전으로 이어졌습니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[2] 4과정 상세
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

냉동 사이클 기준으로 설명하겠습니다:

과정 1→2: 등온 압축 (등온 방열)
- 고온열원(TH)에서 일정한 온도로 열을 방출
- 온도 일정, 엔트로피 감소

과정 2→3: 단열(등엔트로피) 팽창
- 외부와 열교환 없이 팽창
- 온도가 TH에서 TL로 떨어짐
- 엔트로피 일정

과정 3→4: 등온 팽창 (등온 흡열)
- 저온열원(TL)에서 일정한 온도로 열을 흡수
- 이것이 냉동 효과!
- 온도 일정, 엔트로피 증가

과정 4→1: 단열(등엔트로피) 압축
- 외부와 열교환 없이 압축
- 온도가 TL에서 TH로 올라감
- 엔트로피 일정

T-s 선도에서 보면 완벽한 직사각형이 됩니다!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[3] 카르노 COP — 반드시 켈빈(K)!
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

냉동기의 카르노 COP:
COP_carnot = TL / (TH - TL)

히트펌프의 카르노 COP:
COP_HP = TH / (TH - TL)

여기서 T는 반드시 절대온도(K)를 사용해야 합니다!
섭씨를 그대로 넣으면 완전히 틀린 답이 나옵니다.

K = 섭씨 + 273.15 (계산 편의상 + 273도 OK)

참고로:
COP_HP = COP_냉동 + 1
이 관계도 기억해두세요! (에너지 보존에서 유도됨)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[4] 실제 COP vs 카르노 COP 비교
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

실전 예시로 비교해봅시다.

조건: 냉수 7도, 냉각수 35도

카르노 COP 계산:
TL = 7 + 273 = 280 K
TH = 35 + 273 = 308 K
COP_carnot = 280 / (308 - 280) = 280 / 28 = 10.0

실제 냉동기 COP:
- 터보냉동기: 약 5~6
- 스크류냉동기: 약 4~5
- 왕복동냉동기: 약 3~4

카르노 COP 10.0 대비 실제는 절반 이하입니다!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[5] 왜 실제 COP가 카르노보다 낮은가
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

이유 1: 온도차가 더 크다
- 카르노는 냉수/냉각수 온도를 직접 사용
- 실제로는 냉매가 냉수보다 더 낮고(-5도 등),
  냉각수보다 더 높아야(45도 등) 열교환이 됨
- 즉, 실제 TL은 더 낮고 TH는 더 높음 → COP 하락

실제 냉매 온도로 다시 계산:
TL = -5 + 273 = 268 K (증발온도)
TH = 45 + 273 = 318 K (응축온도)
COP_carnot(수정) = 268 / (318 - 268) = 268 / 50 = 5.36
벌써 10에서 5.36으로 떨어졌죠!

이유 2: 비가역 요소
- 압축기 마찰 → 등엔트로피 효율 < 100%
- 팽창밸브 교축 → 등엔탈피이지만 비가역
- 배관 압력손실 → 추가 에너지 소비
- 과열/과냉 → 이상 사이클과의 차이

이유 3: 보조동력
- 냉각수 펌프, 냉각탑 팬, 냉수 펌프 등
- 시스템 COP는 이것까지 포함하면 더 낮아짐

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[6] 실전 예제 — 완전 풀이
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

문제: 냉수 출구 7도, 냉각수 입구 32도인 냉동기가 있다.
증발온도 2도, 응축온도 40도일 때:
(a) 냉수/냉각수 기준 카르노 COP
(b) 실제 냉매 온도 기준 카르노 COP
(c) 실제 COP가 4.5라면 카르노 대비 효율은?

풀이:
(a) TL = 7 + 273 = 280 K, TH = 32 + 273 = 305 K
    COP = 280 / (305 - 280) = 280 / 25 = 11.2

(b) TL = 2 + 273 = 275 K, TH = 40 + 273 = 313 K
    COP = 275 / (313 - 275) = 275 / 38 = 7.24

(c) 카르노 대비 효율 = 실제 COP / 카르노 COP(냉매 기준)
    = 4.5 / 7.24 = 0.621 = 62.1%

카르노 대비 60% 정도면 실무에서 양호한 수준입니다.
고효율 터보냉동기는 70%까지 도달하기도 합니다.

시험에서 자주 묻는 포인트:
"카르노 COP를 높이려면?"
→ TL을 높이거나 TH를 낮추면 됨
→ 즉, 증발온도를 높이고 응축온도를 낮추면 효율 상승
→ 실무: 냉수온도를 높이고, 냉각탑 성능을 개선하라!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Quiz (3문제)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q1. 냉수 5도, 냉각수 30도일 때 냉동기의
    카르노 COP를 구하시오. (반드시 켈빈으로!)

Q2. 카르노 냉동 COP가 8이라면,
    같은 조건에서 카르노 히트펌프 COP는?

Q3. 어떤 냉동기 제조사가 "COP 15를 달성했다"고 주장한다.
    냉수 7도, 냉각수 35도 조건이라면
    이 주장이 타당한지 판단하시오.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Quiz 정답
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

A1. TL = 5 + 273 = 278 K
    TH = 30 + 273 = 303 K
    COP = 278 / (303 - 278) = 278 / 25 = 11.12

A2. COP_HP = COP_냉동 + 1 = 8 + 1 = 9
    (히트펌프는 응축기 방열이 유효 에너지이므로
     압축기 일까지 포함되어 항상 1만큼 더 크다)

A3. 타당하지 않다.
    TL = 280 K, TH = 308 K
    카르노 COP = 280 / 28 = 10.0
    카르노 COP는 이론적 최대값이므로
    어떤 실제 냉동기도 10.0을 넘을 수 없다.
    COP 15는 열역학 제2법칙에 위배되는 주장이다!
    (측정 조건이나 계산 방법에 오류가 있을 가능성이 높다)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

카르노 사이클은 "이보다 더 좋을 순 없다"의 기준입니다.
실제 냉동기의 효율을 평가하고 개선 방향을 찾는 데
카르노 COP가 훌륭한 벤치마크가 되어줍니다.
