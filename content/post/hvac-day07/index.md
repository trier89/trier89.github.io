HVAC 열역학 스터디 — Day 7
Week 1 심화: 실무 계산 연습

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

오늘은 실전 계산력을 키우는 날입니다!
공조냉동기술사 시험에서도 계산 문제는 배점이 높으니까,
공식을 외우는 것보다 "어떤 상황에 어떤 공식을 쓰는지"
판단하는 연습이 중요합니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[1] 냉동능력 계산 — 냉수 유량과 온도차
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

기본 공식:
Q = m x c x DT

여기서
- Q: 열량 (kW)
- m: 질량유량 (kg/s)
- c: 물의 비열 = 4.186 kJ/(kg.K)
- DT: 온도차 (K 또는 도C — 차이이므로 같음)

실무 팁:
냉수 유량이 L/min으로 주어지면?
→ kg/s로 환산: L/min ÷ 60 = kg/s (물 밀도 ≒ 1 kg/L)

간편 공식도 기억하세요:
Q(kcal/h) = 유량(L/min) x 60 x DT
Q(kW) = 유량(L/min) x 4.186 x DT / 60

USRT 환산:
1 USRT = 3.517 kW = 3,024 kcal/h

예제) 냉수 유량 200 L/min, 입구 12도, 출구 7도
Q = 200 x 4.186 x 5 / 60 = 69.8 kW = 19.8 USRT

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[2] 열관류율 계산 — 다층 벽체
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

열관류율 U값 구하는 순서:
(1) 각 층의 열저항 R = 두께(m) / 열전도율 k
(2) 표면 열전달저항 추가 (실내측 Ri, 실외측 Ro)
(3) 전체 열저항 Rt = Ri + R1 + R2 + ... + Ro
(4) U = 1 / Rt

실무에서 자주 쓰는 표면 열전달저항:
- 실내측 (수직면): 0.11 (m2.K)/W
- 실외측 (수직면): 0.04 (m2.K)/W
- 실내측 (천장/상향): 0.09 (m2.K)/W

예제) 외벽 구성
- 실내측: 0.11
- 석고보드 12mm, k = 0.17 → R = 0.012/0.17 = 0.071
- 단열재 85mm, k = 0.034 → R = 0.085/0.034 = 2.500
- 콘크리트 200mm, k = 1.6 → R = 0.200/1.6 = 0.125
- 실외측: 0.04

Rt = 0.11 + 0.071 + 2.500 + 0.125 + 0.04 = 2.846
U = 1 / 2.846 = 0.351 W/(m2.K)

여기서 단열재의 열저항이 전체의 약 88%를 차지합니다.
단열재가 얼마나 중요한지 숫자로 확인되죠!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[3] 현열비(SHF) 계산
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

SHF = Qs / Qt = 현열 / (현열 + 잠열)

현열(Qs): 온도를 변화시키는 열
잠열(Ql): 습도를 변화시키는 열 (수증기 응축/증발)

예제) 어떤 사무실의 부하가 다음과 같을 때
- 현열부하: 인체 3,500W + 조명 5,000W + 기기 4,000W
           + 외피 8,000W + 외기 6,000W = 26,500W
- 잠열부하: 인체 2,500W + 외기 4,000W = 6,500W

SHF = 26,500 / (26,500 + 6,500) = 26,500 / 33,000 = 0.80

SHF 0.80이면 일반적인 사무실 수준입니다.
습기가 많은 식당이면 SHF 0.6~0.7 정도로 내려갑니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
실전 계산 문제 3개
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

문제 1) 냉동능력 계산
100 USRT 냉동기가 냉수를 7도에서 12도로 만들어 공급한다.
이 냉동기에 필요한 냉수 유량은 몇 L/min인가?

문제 2) 열관류율 + 열손실
외벽 열관류율이 U = 0.35 W/(m2.K)이고 면적이 50 m2이다.
실내 22도, 실외 -10도일 때 이 벽체를 통한 열손실은?

문제 3) 현열비 + 풍량
실내 현열부하 30 kW, 잠열부하 10 kW인 공간이 있다.
(a) SHF는?
(b) 급기온도 15도, 실내온도 26도일 때 필요 풍량(CMH)은?
    (공기 비열: 1.006 kJ/(kg.K), 밀도: 1.2 kg/m3)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
풀이
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

풀이 1)
100 USRT = 100 x 3.517 = 351.7 kW
Q = m x c x DT 에서
351.7 = m x 4.186 x 5
m = 351.7 / 20.93 = 16.8 kg/s
L/min 환산: 16.8 x 60 = 1,008 L/min

실무 간편식: USRT x 10 ≒ L/min (DT 5도 기준)
100 x 10 = 약 1,000 L/min → 거의 일치!

풀이 2)
Q = U x A x DT
Q = 0.35 x 50 x (22 - (-10))
Q = 0.35 x 50 x 32
Q = 560 W = 0.56 kW

벽체 하나에서 560W가 빠져나가는 겁니다.
건물 전체로 보면 외피 열손실이 상당히 큰 비중을 차지하죠.

풀이 3)
(a) SHF = 30 / (30 + 10) = 30 / 40 = 0.75

(b) 현열부하 기준으로 풍량 산정:
    Qs = m x c x DT
    30 = m x 1.006 x (26 - 15)
    30 = m x 11.066
    m = 2.712 kg/s

    체적유량 = m / 밀도 = 2.712 / 1.2 = 2.26 m3/s
    CMH = 2.26 x 3600 = 8,136 CMH

    약 8,100 CMH의 급기가 필요합니다.
