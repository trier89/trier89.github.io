🧱 Day 05 — 열관류와 U값

어제 전도, 대류, 복사를 배웠는데요,
오늘은 이것들이 합쳐진 '열관류'와 실무에서 핵심인 U값을 정리합니다.


━━━━━━━━━━━━━━━━━━━━
1. 열관류(열통과)란?
━━━━━━━━━━━━━━━━━━━━

열관류 = 대류 + 전도 + 대류가 연속으로 일어나는 열전달

건물 외벽을 예로 들면:
실외 공기(대류) → 외벽(전도) → 실내 공기(대류)

이렇게 유체 → 고체 → 유체를 거쳐 열이 '관통'하는 과정을
열관류(Overall Heat Transfer)라고 합니다.

비유: 열이 여러 장벽을 통과하는 릴레이 경주 같은 거예요.
가장 느린 구간이 전체 속도를 결정합니다.


━━━━━━━━━━━━━━━━━━━━
2. 열관류율 U값
━━━━━━━━━━━━━━━━━━━━

U값 (W/m²·K) = 1m² 면적에서 1K 온도차당 이동하는 열량

U값이 크다 → 열이 잘 통과한다 → 단열 성능이 나쁘다
U값이 작다 → 열이 잘 안 통과한다 → 단열 성능이 좋다

열관류 열전달 공식:
Q = U × A × ΔT

Q: 열량 (W)
U: 열관류율 (W/m²·K)
A: 면적 (m²)
ΔT: 실내외 온도차 (K)

예제) 외벽 면적 100m², U값 0.5 W/m²·K
실내 26°C, 실외 35°C일 때:
Q = 0.5 × 100 × (35-26) = 450 W
→ 이 벽을 통해 450W의 열이 침입합니다.


━━━━━━━━━━━━━━━━━━━━
3. 열저항 R값과의 관계
━━━━━━━━━━━━━━━━━━━━

R = 1/U (m²·K/W)

R값은 열에 대한 '저항'입니다.
전기로 비유하면: U값 = 전류, R값 = 저항

R값이 크다 → 단열이 잘 된다 → 좋은 것!

여러 층으로 된 벽체의 R값은 더하면 됩니다:
R전체 = R실내대류 + R재료1 + R재료2 + ... + R실외대류

각 재료층의 R = 두께(L) / 열전도율(k)

예) 콘크리트 200mm + 단열재 50mm 벽체:
R콘크리트 = 0.2 / 1.4 = 0.143
R단열재 = 0.05 / 0.04 = 1.25
R실내대류 ≈ 0.11
R실외대류 ≈ 0.04

R전체 = 0.11 + 0.143 + 1.25 + 0.04 = 1.543
U = 1 / 1.543 = 0.648 W/m²·K

50mm 단열재 하나가 R값의 80%를 차지합니다!
단열재가 얼마나 효과적인지 확인할 수 있죠.


━━━━━━━━━━━━━━━━━━━━
4. 건축에서 단열 성능 평가
━━━━━━━━━━━━━━━━━━━━

건축물 에너지절약설계기준에서는
부위별로 U값 기준을 정해 놓고 있습니다.

예시 (중부지역 기준, 참고용):
- 외벽: 0.240 W/m²·K 이하
- 지붕: 0.150 W/m²·K 이하
- 창호: 1.000 W/m²·K 이하

창문의 U값이 벽보다 훨씬 높죠?
→ 건물에서 열이 가장 많이 새는 곳은 창문입니다.
그래서 이중창, 삼중창, Low-E 유리를 쓰는 거예요.


━━━━━━━━━━━━━━━━━━━━
5. 열교환기와 U값
━━━━━━━━━━━━━━━━━━━━

열교환기에서도 U값은 핵심 성능 지표입니다.

열교환기 용량: Q = U × A × LMTD

LMTD: 대수평균온도차 (로그 평균)

열교환기는 벽체와 반대로, U값이 클수록 좋습니다!
같은 면적에서 더 많은 열을 교환할 수 있으니까요.


━━━━━━━━━━━━━━━━━━━━
6. 파울링과 U값 저하
━━━━━━━━━━━━━━━━━━━━

냉동기 실무에서 매우 중요한 개념입니다.

파울링(Fouling) = 열교환 표면에 이물질이 쌓이는 것

응축기: 냉각수 속 스케일, 조류 → 튜브 내벽에 부착
증발기: 냉수 속 이물질 → 튜브 표면에 부착

이물질 층이 추가 열저항을 만들어:
R전체 증가 → U값 감소 → 열교환 효율 저하

결과:
- 응축기 파울링 → 응축온도/압력 상승 → 전력 소비 증가
- 증발기 파울링 → 증발온도 하락 → 냉방능력 감소

그래서 냉동기 정기 점검에서 튜브 세관(청소)이
중요한 유지보수 항목인 거예요!

접근온도(Approach Temperature)를 모니터링하면
파울링 진행 정도를 파악할 수 있습니다.
