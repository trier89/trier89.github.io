━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Day 14 - Week 2 심화: 카르노 COP 실전 계산
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

안녕하세요! 오늘은 카르노 COP를 다양한 조건에서 직접 계산하고, 실제 냉동기와 비교해보는 시간입니다. 기술사 시험에서 계산 문제가 나올 때 당황하지 않도록 손에 익혀봅시다.

━━━ 카르노 COP 공식 복습 ━━━

냉동기: COP_R = T_L / (T_H - T_L)
히트펌프: COP_HP = T_H / (T_H - T_L)

여기서 T는 반드시 절대온도(K)를 사용합니다!
섭씨 → 켈빈: T(K) = T(도C) + 273.15 (계산 편의상 +273 사용)

중요한 관계: COP_HP = COP_R + 1 (항상 성립!)

━━━ 조건별 카르노 COP 비교 ━━━

카르노 COP는 온도 조건에 따라 크게 달라집니다.

(1) 일반 냉방: 증발 7도C / 응축 35도C
    COP_R = 280 / 28 = 10.0

(2) 저온 냉동: 증발 -25도C / 응축 40도C
    COP_R = 248 / 65 = 3.82

(3) 극저온 냉동: 증발 -40도C / 응축 45도C
    COP_R = 233 / 85 = 2.74

온도차가 커질수록 COP가 급격히 떨어지는 게 보이시죠? 이것이 저온 냉동이 에너지를 많이 먹는 근본적인 이유입니다.

━━━ 실제 COP vs 카르노 COP ━━━

실제 냉동기는 카르노의 40~60% 수준입니다.

예시) 증발 -10도C, 응축 40도C 조건
- 카르노 COP = 263 / 50 = 5.26
- 실제 냉동기 COP ≒ 2.5~3.0 (카르노의 약 50~57%)

카르노 대비 효율 = 실제COP / 카르노COP x 100%

이 비율을 "카르노 효율" 또는 "2nd law efficiency"라고 부릅니다. 기술사 답안에서 이 개념을 쓰면 좋은 점수를 받을 수 있어요.

━━━ 엔탈피로 압축기 동력 계산 ━━━

실제 시스템에서는 P-h 선도의 엔탈피 값으로 계산합니다.

압축기 동력 W = m x (h2 - h1)
- m: 냉매 질량유량 (kg/s)
- h1: 압축기 입구 엔탈피
- h2: 압축기 출구 엔탈피

냉동능력 Q_e = m x (h1 - h4)
실제 COP = Q_e / W = (h1 - h4) / (h2 - h1)

이 공식은 Day 18에서 더 자세히 다룰 예정이니, 오늘은 흐름만 잡아두세요.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
실전 문제 (3문제)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[문제 1]
냉동 창고(증발온도 -30도C, 응축온도 35도C)의 카르노 COP를 구하고, 실제 COP가 2.0일 때 카르노 효율(제2법칙 효율)을 계산하시오.

[문제 2]
히트펌프로 난방 시 외기온도 -5도C(저열원), 실내 급기온도 45도C(고열원)일 때:
(a) 카르노 COP_HP를 구하시오.
(b) 이 히트펌프의 실제 COP가 3.2일 때, 10kW 난방에 필요한 압축기 소비전력은?

[문제 3]
R-134a 냉동기에서 다음 엔탈피 값이 주어졌을 때:
h1(압축기 입구) = 395 kJ/kg
h2(압축기 출구) = 430 kJ/kg
h4(증발기 입구) = 250 kJ/kg
냉매 순환량 0.05 kg/s일 때:
(a) 냉동능력(kW)
(b) 압축기 동력(kW)
(c) COP를 구하시오.

━━━ 풀이 ━━━

[풀이 1]
T_L = -30 + 273 = 243K
T_H = 35 + 273 = 308K
카르노 COP = 243 / (308 - 243) = 243 / 65 = 3.74

카르노 효율 = 실제COP / 카르노COP = 2.0 / 3.74 = 0.535 = 53.5%

실제 냉동 창고가 이론 한계의 약 54%로 운전되고 있다는 의미입니다.

[풀이 2]
(a) T_L = -5 + 273 = 268K, T_H = 45 + 273 = 318K
COP_HP = 318 / (318 - 268) = 318 / 50 = 6.36

(b) COP = Q_H / W 이므로
W = Q_H / COP = 10 / 3.2 = 3.125kW

3.125kW 전기로 10kW 난방 가능! 이것이 히트펌프의 매력이죠.

[풀이 3]
(a) 냉동능력 = m x (h1 - h4) = 0.05 x (395 - 250) = 0.05 x 145 = 7.25kW
(b) 압축기 동력 = m x (h2 - h1) = 0.05 x (430 - 395) = 0.05 x 35 = 1.75kW
(c) COP = 7.25 / 1.75 = 4.14
