HVAC 열역학 스터디 — Day 9
일과 열의 관계, 내부에너지

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

어제 제1법칙에서 Q = DU + W를 배웠죠.
오늘은 이 세 요소 중 "일(W)"과 "내부에너지(U)"를
좀 더 깊이 들여다봅니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[1] 기계적 일 vs 열역학적 일
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

기계적 일 (역학에서 배운 것):
- W = F x d (힘 x 거리)
- 예: 펌프가 물을 밀어올리는 일

열역학적 일 (열역학에서의 일):
- 기체가 팽창하면서 피스톤을 밀어내는 것
- W = P x DV (압력 x 체적 변화) — 정압일 경우
- 기체가 팽창(DV > 0) → 계가 일을 함(W > 0)
- 기체가 압축(DV < 0) → 외부가 계에 일을 함(W < 0)

핵심 차이:
역학에서의 일은 눈에 보이는 운동이지만,
열역학에서의 일은 기체 분자들이 벽면을 미는 것까지 포함.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[2] P-V 선도에서의 일
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

P-V 선도(압력-체적 그래프)에서
곡선 아래의 면적 = 일의 양

이게 왜 중요하냐면,

같은 시작점에서 같은 끝점까지 가더라도
경로가 다르면 일의 양이 달라집니다!
(열과 일은 경로함수, 내부에너지는 상태함수)

상태함수 vs 경로함수:
- 상태함수: 시작과 끝 상태만 알면 됨 (U, H, T, P)
- 경로함수: 어떤 과정을 거쳤는지가 중요 (Q, W)

시험에 자주 나오는 개념이니 꼭 구분하세요!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[3] 내부에너지란
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

내부에너지(U)는 물질을 구성하는 분자들의
운동에너지와 위치에너지의 총합입니다.

쉽게 말하면:
- 분자들이 빠르게 움직이면(온도 높으면) → U 증가
- 분자들 사이 인력이 강하면 → 위치에너지도 포함

이상기체의 경우:
- 내부에너지는 오직 온도에만 의존
- U = f(T) — 온도만 같으면 내부에너지도 같음
- DU = m x Cv x DT (Cv: 정적비열)

실제 기체나 액체는 압력이나 체적에도 약간 영향받지만,
이상기체라고 가정하면 계산이 훨씬 간단해집니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[4] 비내부에너지 (u, kJ/kg)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

비(specific) = "단위 질량당"이라는 뜻

- U: 전체 내부에너지 (kJ)
- u: 비내부에너지 = U / m (kJ/kg)

냉동 분야에서는 냉매 1kg 기준으로 계산하는 경우가 많아서
비내부에너지(u)를 자주 씁니다.

예를 들어 냉매 R-134a의 상태표를 보면:
- 포화액 상태 (0도): u_f ≒ 200 kJ/kg
- 포화증기 상태 (0도): u_g ≒ 378 kJ/kg
- 차이(u_fg) ≒ 178 kJ/kg → 증발에 필요한 내부에너지

이 값들은 냉매 물성표에서 직접 찾아 쓰는 거예요.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[5] 냉동 실무 적용 — 압축기 동력의 기초
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

냉동 사이클에서 압축기가 하는 일:

W_comp = m x (h2 - h1)

여기서 h(엔탈피)를 쓰는 이유는
압축기가 개방계이기 때문입니다.
(내일 엔탈피를 자세히 다룰 거예요)

그런데 본질적으로 보면:
엔탈피 = 내부에너지 + 유동일
h = u + Pv

즉, 압축기 동력 계산의 뿌리에는
오늘 배운 내부에너지와 일의 개념이 깔려 있습니다.

실무 예시:
R-410A 냉매, 질량유량 0.5 kg/s
압축기 입구 h1 = 420 kJ/kg
압축기 출구 h2 = 460 kJ/kg

W = 0.5 x (460 - 420) = 20 kW

이 20 kW가 압축기 모터가 소비하는 이론 동력입니다.
실제로는 기계효율, 전기효율 등을 곱해야 하지만
기본 구조는 이 공식 하나입니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Quiz (3문제)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q1. 열(Q)과 일(W)은 상태함수인가 경로함수인가?
    내부에너지(U)는?

Q2. 이상기체의 내부에너지는 어떤 변수에만 의존하는가?
    DU를 구하는 공식은?

Q3. 같은 초기 상태에서 같은 최종 상태로 가는 두 과정이
    있을 때, 두 과정의 DU는 같은가 다른가?
    W는 같은가 다른가?

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Quiz 정답
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

A1. Q와 W는 경로함수이다.
    과정(경로)에 따라 값이 달라진다.
    U는 상태함수이다.
    초기 상태와 최종 상태만 알면 DU가 결정된다.

A2. 이상기체의 내부에너지는 온도(T)에만 의존한다.
    DU = m x Cv x DT
    (Cv: 정적비열, 정적과정이 아니어도 이상기체에서는 성립)

A3. DU는 같다 (상태함수이므로).
    W는 다를 수 있다 (경로함수이므로).
    마찬가지로 Q도 다를 수 있다.
    하지만 Q - W = DU는 항상 성립 (제1법칙).
