💧 Day 03 — 현열과 잠열

오늘은 냉동공조의 핵심 중의 핵심!
'현열'과 '잠열'의 차이를 확실히 잡아봅니다.


━━━━━━━━━━━━━━━━━━━━
1. 현열 (Sensible Heat)
━━━━━━━━━━━━━━━━━━━━

현열은 '온도가 변하는 열'입니다.

물을 10°C에서 90°C로 가열하면?
→ 온도가 올라갑니다. 이때 들어간 열이 현열.

온도계로 측정할 수 있어서 '느낄 수 있는 열' = 현열
(현 顯 = 나타날 현)

공식: Q = mcΔT (어제 배운 그 공식!)


━━━━━━━━━━━━━━━━━━━━
2. 잠열 (Latent Heat)
━━━━━━━━━━━━━━━━━━━━

잠열은 '상태가 변하는 열'입니다.

얼음 0°C → 물 0°C: 온도는 안 변하는데 열이 들어감!
물 100°C → 수증기 100°C: 역시 온도 변화 없이 열 흡수!

온도계에 안 잡히는 '숨어있는 열' = 잠열
(잠 潛 = 숨을 잠)

대표적인 잠열값:
- 얼음의 융해잠열: 334 kJ/kg
- 물의 증발잠열: 2,257 kJ/kg (어마어마하게 큽니다!)

비유: 얼음이 녹을 때를 생각해 보세요.
주변에서 열을 쏙쏙 빨아들이지만 얼음물의 온도는
0°C에서 움직이지 않죠? 이것이 잠열의 힘입니다.


━━━━━━━━━━━━━━━━━━━━
3. 왜 공조에서 잠열이 중요한가?
━━━━━━━━━━━━━━━━━━━━

여름철 냉방할 때 에어컨이 처리하는 부하:

현열 부하: 실내 온도를 낮추는 것
잠열 부하: 실내 습기를 제거하는 것 (제습)

습한 공기가 냉각코일을 지나면
공기 중 수증기가 물방울로 변합니다 (응축).
이때 수증기가 잠열을 방출하고 → 코일이 그 열을 흡수합니다.

실제로 여름철 냉방부하의 30~40%가 잠열입니다!
온도만 낮추면 될 것 같지만, 습도 처리에 엄청난 에너지가 들어요.


━━━━━━━━━━━━━━━━━━━━
4. 증발잠열과 냉동사이클
━━━━━━━━━━━━━━━━━━━━

냉동사이클의 핵심 원리:

냉매가 증발기에서 액체 → 기체로 변할 때
주변에서 증발잠열을 흡수합니다 → 이것이 냉방 효과!

물의 증발잠열이 2,257 kJ/kg인 것처럼
R-410A 냉매도 약 220 kJ/kg의 증발잠열을 가집니다.

요약하면:
증발기: 냉매가 증발하면서 열 흡수 (냉방)
응축기: 냉매가 응축하면서 열 방출 (실외기가 뜨거운 이유)


━━━━━━━━━━━━━━━━━━━━
5. 현열비 (SHF, Sensible Heat Factor)
━━━━━━━━━━━━━━━━━━━━

SHF = 현열 / 전체 열(현열 + 잠열)

SHF가 높다 (0.8~1.0) → 온도 부하가 대부분 (사무실, 전산실)
SHF가 낮다 (0.5~0.7) → 습기 부하가 많다 (수영장, 주방, 강당)

공조기를 선정할 때 SHF에 따라
코일 설계, 풍량, 냉수온도가 달라집니다.
같은 냉방용량이라도 SHF가 낮으면 더 낮은 냉수온도가 필요해요.
