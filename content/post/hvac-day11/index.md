HVAC 열역학 스터디 — Day 11
열역학 제2법칙과 엔트로피

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

제1법칙이 "에너지의 양"을 다뤘다면,
제2법칙은 "에너지의 방향"을 다룹니다.
왜 냉동기에 전기가 필요한지, 오늘 명쾌하게 이해할 수 있어요!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[1] 제2법칙 — 열은 고온에서 저온으로
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

클라우지우스 표현:
"열은 자연적으로 저온체에서 고온체로 이동할 수 없다."

켈빈-플랑크 표현:
"하나의 열원에서 열을 받아 전부 일로 바꾸는 것은 불가능하다."

두 표현은 다르게 보이지만 수학적으로 동등합니다.
시험에서는 두 표현 모두 알아야 합니다!

실생활 예시:
- 뜨거운 커피는 식지만, 식은 커피가 저절로 뜨거워지진 않음
- 냉장고 안은 차갑지만, 전기 없이는 차가워질 수 없음
- 열은 방향성이 있다!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[2] 엔트로피란
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

엔트로피(S)를 한마디로: 무질서도, 비가역성의 척도

수학적 정의:
dS >= dQ / T (등호는 가역과정일 때)

비엔트로피: s (kJ/(kg.K))

직관적 이해:
- 얼음이 녹으면 → 분자 배열이 무질서해짐 → 엔트로피 증가
- 기체가 팽창하면 → 분자가 넓은 공간에 퍼짐 → 엔트로피 증가
- 자연에서 일어나는 모든 과정 → 전체 엔트로피 증가

핵심 원리:
고립계(우주 전체)의 엔트로피는 항상 증가하거나 유지된다.
절대로 감소하지 않는다!

DS_total >= 0

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[3] 클라우지우스 부등식
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

순환 과정에서:
적분(dQ/T) <= 0

등호: 가역 과정 (이상적)
부등호: 비가역 과정 (실제)

이것이 의미하는 바:
실제 과정에서는 항상 엔트로피가 생성됩니다.
마찰, 열전달 온도차, 유체 혼합 등이 비가역성의 원인이죠.

시험 팁:
"엔트로피 생성 = 0이면 가역, > 0이면 비가역"
이 한 줄이 엔트로피 관련 문제의 핵심 판단 기준입니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[4] 냉동기는 왜 전기가 필요한가
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

냉동기의 목적:
저온(실내)에서 열을 빼내어 고온(실외)으로 버리는 것

이것은 열의 자연스러운 방향과 반대입니다!
저온 → 고온으로 열을 옮기려면 반드시 외부 일(전기)이 필요.

이것이 바로 제2법칙입니다!

에너지 흐름:
실내(저온) --[Qe]--> 냉동기 --[Qc]--> 실외(고온)
                        ^
                        |
                    전기(W)

Qc = Qe + W (제1법칙)
W가 반드시 필요 (제2법칙)

만약 제2법칙이 없다면?
전기 없이 열을 퍼올릴 수 있는 "무한 냉동기"가 가능하겠지만,
자연은 그것을 허락하지 않습니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[5] 가역과정 vs 비가역과정
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

가역과정 (Reversible):
- 과정을 역방향으로 돌려도 흔적이 남지 않는 이상적 과정
- 엔트로피 생성 = 0
- 현실에는 존재하지 않음 (이론적 최선)
- 예: 마찰 없는 단열 압축, 무한히 작은 온도차의 열전달

비가역과정 (Irreversible):
- 실제 세계의 모든 과정
- 엔트로피 생성 > 0
- 비가역성의 원인들:
  (1) 마찰 (기계적 손실)
  (2) 열전달 온도차 (열교환기에서 온도차가 클수록 비가역)
  (3) 유체 혼합 (다른 온도의 유체 섞기)
  (4) 급격한 팽창/압축 (교축 = 팽창밸브)
  (5) 전기저항에 의한 발열

냉동 사이클에서 비가역 요소:
- 압축기: 마찰, 열손실 → 등엔트로피 효율 < 100%
- 팽창밸브: 교축 과정 자체가 비가역 (등엔탈피이지만 엔트로피 증가)
- 열교환기: 온도차가 있어야 열전달 → 비가역

이 비가역 요소들 때문에 실제 COP가 이론보다 낮아집니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Quiz (3문제)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q1. 클라우지우스 표현과 켈빈-플랑크 표현을
    각각 한 문장으로 쓰시오.

Q2. 팽창밸브는 등엔탈피 과정이다. 그렇다면 엔트로피는
    증가하는가, 감소하는가, 변화없는가? 그 이유는?

Q3. 어떤 압축기의 등엔트로피 효율이 80%라는 것은
    어떤 의미인가? 실제 압축일은 이상적인 경우보다
    크게 되는가 작게 되는가?

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Quiz 정답
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

A1. 클라우지우스: 열은 자연적으로 저온체에서 고온체로
    이동할 수 없다 (외부 일 없이는).
    켈빈-플랑크: 하나의 열원에서 열을 흡수하여 그 전부를
    일로 변환하는 것은 불가능하다.

A2. 엔트로피는 증가한다.
    팽창밸브에서의 교축(throttling)은 대표적인 비가역 과정이다.
    엔탈피는 보존되지만, 압력 강하로 인해 일부 액체가
    증발(플래시 가스)하면서 분자의 무질서도가 증가한다.
    비가역 과정이므로 엔트로피는 반드시 증가한다.

A3. 등엔트로피 효율 80%란, 이상적(가역 단열) 압축에
    필요한 일 대비 실제 압축에 필요한 일의 비율이다.
    효율 = 이상 압축일 / 실제 압축일 = 0.80
    따라서 실제 압축일 = 이상 압축일 / 0.80 으로
    실제 압축일이 이상적인 경우보다 크다 (약 25% 더 큼).
    마찰과 열손실 때문에 더 많은 에너지가 필요한 것이다.
