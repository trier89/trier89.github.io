HVAC 열역학 스터디 — Day 6
Week 1 복습: 핵심 정리 + 종합 퀴즈

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

안녕하세요! 오늘은 Day 1~5에서 배운 내용을 빠르게 정리하고,
종합 퀴즈로 마무리하는 날입니다.
실무에서 자주 쓰이는 개념만 골라서 핵심 한 줄로 압축했어요.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[1] 온도와 열
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

핵심: 온도는 분자 운동의 활발함 정도, 열은 온도 차이에 의해 이동하는 에너지.

- 온도가 같으면 열 이동 없음 (열평형 = 제0법칙)
- 절대온도 K = 섭씨 + 273.15
- 실무에서는 온도"차"가 중요 (냉수 입출구 차이 = 5도 등)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[2] 비열과 열량 (Q = mcDT)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

핵심: 물질 1kg을 1도 올리는 데 필요한 열량이 비열(c).

- 물의 비열: 4.186 kJ/(kg.K) — 거의 모든 공조 계산의 기본
- Q = m x c x DT (질량 x 비열 x 온도차)
- 냉동톤 환산: 1 USRT = 3.517 kW = 3,024 kcal/h

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[3] 현열과 잠열 (SHF)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

핵심: 현열은 온도 변화, 잠열은 상태 변화에 필요한 열.

- 현열비(SHF) = 현열 / (현열 + 잠열)
- SHF가 높으면 → 건조한 환경 (사무실 등)
- SHF가 낮으면 → 습한 환경 (수영장, 식당 주방 등)
- 공조 설계 시 SHF에 따라 코일 선정이 달라짐

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[4] 열전달 3가지 (전도/대류/복사)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

핵심: 열은 전도, 대류, 복사 세 가지 방법으로 이동한다.

- 전도: 물질 내부 (벽체, 배관 등) → 열전도율 k [W/(m.K)]
- 대류: 유체 표면 (공기-벽 사이) → 열전달계수 h [W/(m2.K)]
- 복사: 전자기파 (태양열, 적외선) → 흑체 기준 스테판-볼츠만 법칙
- 건물 외피 열부하 계산 = 전도 + 대류 + 복사 모두 고려

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[5] 열관류와 U값
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

핵심: 열관류 = 전도 + 대류를 통합한 복합 열전달.

- 열관류율 U [W/(m2.K)] = 1 / (1/hi + L/k + 1/ho)
- U값이 클수록 → 열이 잘 빠져나감 → 단열 성능 나쁨
- 건축법 단열 기준은 "열관류율 U값 이하"로 규정
- 서울 기준 외벽 U값: 0.240 W/(m2.K) 이하 (에너지절약설계기준)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
종합 퀴즈 (5문제)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q1. 냉수량 100 L/min, 입구 12도, 출구 7도일 때 냉동능력은
    몇 kW인가? (물의 비열 4.186 kJ/(kg.K), 밀도 1 kg/L)

Q2. 사무실의 현열비(SHF)가 0.85라면, 이 공간은 상대적으로
    건조한 편인가 습한 편인가? 그 이유는?

Q3. 열전도율 k = 0.04 W/(m.K)인 단열재 50mm를 사용할 때,
    단열재만의 열저항 R은 얼마인가?

Q4. "열은 항상 고온에서 저온으로 이동한다"는 열역학 몇 법칙과
    관련되며, 냉동기가 존재하는 이유를 간단히 설명하시오.

Q5. 외벽 구성이 아래와 같을 때 열관류율 U값을 구하시오.
    - 실내측 표면 열전달저항: 0.11 (m2.K)/W
    - 콘크리트 150mm, k = 1.6 W/(m.K)
    - 단열재 100mm, k = 0.035 W/(m.K)
    - 실외측 표면 열전달저항: 0.04 (m2.K)/W

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
퀴즈 정답
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

A1. Q = m x c x DT
    m = 100 L/min = 100 kg/min
    Q = 100 x 4.186 x (12 - 7) = 2,093 kJ/min
    kW로 환산: 2,093 / 60 = 34.9 kW
    (약 9.9 USRT)

A2. 건조한 편이다.
    SHF 0.85는 전체 열부하 중 85%가 현열(온도 변화)이고
    잠열(습도 변화)은 15%에 불과하다는 뜻이다.
    잠열 비중이 낮으면 습기 발생이 적은 공간이다.

A3. R = L / k = 0.05 / 0.04 = 1.25 (m2.K)/W
    단열재 두께 50mm = 0.05m로 환산하는 것을 잊지 말 것!

A4. 열역학 제2법칙과 관련된다.
    자연 상태에서 열은 고온→저온으로만 흐른다.
    저온→고온으로 열을 옮기려면 외부에서 일(전기 등)을
    가해야 하며, 이것이 바로 냉동기의 원리이다.

A5. 전체 열저항 Rt = 0.11 + (0.15/1.6) + (0.1/0.035) + 0.04
    = 0.11 + 0.094 + 2.857 + 0.04 = 3.101 (m2.K)/W
    U = 1 / 3.101 = 0.323 W/(m2.K)
    (참고: 서울 기준 0.240 이하이므로 이 벽체는 기준 미달!)
